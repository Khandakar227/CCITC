A blog is a discussion or informational website published on the World Wide Web consisting of discrete, often informal diary-style text entries (posts). Members and Executives can upload their contents here. If you have any projects, contents, or articles, try submitting them here. Show them to the world. We will look into your posts and upload them on the site. If you have any questions regarding it, please contact the club executives.

Note:Use HTML markdowns

You can add more than some plain texts and image, You can also use html tags to make your blogs look interesting and attractive. Or you can just write normally and let us do the tagging.

If don't know about html tags, it's alright. Here are some basic html tags(along with some custom tags of our own) that you can use.

  • <h2> "WRITE SUB-HEADING HERE" </h2>
  • <p> "WRITE PARAGRAPH HERE" </p>
  • <code> "IF YOUR BLOG CONTAINS ANY CODES. WRITE PROGRAMMING CODES/ SYNTAX HERE USE IT FOR EACH LINES OF CODE" </code>
  • <codeContainer> "A CONTAINER OR A BLOCK THAT CONTAINS ALL YOUR CODE IF YOU HAVE MULTIPLE LINES OF CODE. WRITE ALL YOUR CODES ALONG WITH THE <code></code> FOR EACH LINE " </codeContainer>
  • <strong> "TEXTS BETWEEN THIS TAG BECOMES BOLDER" </strong>
  • <br> TEXTS AFTER THIS TAG ARE ADDED IN THE NEXT LINE(In short breaks the line)
  • <span class="redtxt" > "TEXTS BETWEEN THIS TAG BECOMES RED IN COLOR" </span>
  • <strong> "TEXTS BETWEEN THIS TAG BECOMES BOLDER" </strong>
  • <li> "USE IT TO LIST ITEMS. USE IT FOR EVERY ITEMS YOU PREFER TO BE LISTED." </li>
  • <sup> "DEFINES SUPERSCRIPTED TEXT." </sup> example: X<sup>2</sup> = X2
  • <sub> "DEFINES SUBSCRIPTED TEXT." </sub> example: CO<sub>2</sub> = CO2
